# 📊 Olist Marketplace Analytics Dashboard

End-to-end аналитический проект по анализу маркетплейса Olist.

## 🚀 Обзор проекта

В рамках проекта выполнен полный цикл аналитической работы:

- исследование данных
- построение SQL-витрин
- расчет бизнес-метрик
- сегментация клиентов
- анализ качества сервиса
- разработка интерактивного Power BI Dashboard

## 🔍 Основные выводы

- Более 94% клиентов совершают только одну покупку.
- Repeat Purchase Rate составляет 5.96%.
- Основной потенциал роста находится в удержании клиентов.
- Задержки доставки негативно влияют на оценки покупателей.
- Несколько категорий генерируют основную часть GMV.
